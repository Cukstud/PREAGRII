# 🌱 PREAGRI - Precision Agriculture System

PREAGRI is a Machine Learning-based Precision Agriculture system designed to assist farmers in making data-driven decisions. The system integrates intelligent modules to improve crop productivity and reduce losses.

---

## 🚀 Features

### 1️⃣ Crop Prediction
Predicts the most suitable crop based on:
- Soil nutrients (N, P, K)
- Temperature
- Humidity
- pH level
- Rainfall

### 2️⃣ Fertilizer Recommendation
Suggests optimal fertilizers by analyzing:
- Soil nutrient deficiencies
- Target crop requirements

### 3️⃣ Leaf Disease Detection
Detects plant diseases using image classification:
- Upload leaf image
- Model predicts disease type
- Helps in early prevention and treatment

---

## 🛠️ Tech Stack

- Python
- Scikit-learn
- Pandas & NumPy
- TensorFlow / CNN (for disease detection)
- Flask (for deployment)
- HTML, CSS (Frontend)

---

## 📊 Dataset Used

- Crop Recommendation Dataset
- Fertilizer Dataset
- Plant Leaf Disease Dataset

---

## 🎯 Objective

To enable smart farming practices using Machine Learning by providing:
- Accurate crop recommendations
- Cost-efficient fertilizer usage
- Early disease identification
